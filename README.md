## 사용 기술

### 프론트엔드

- Html/Css
- Javascript, JQuery
- Bootstrap 5
- Thymeleaf

### 백엔드

- Java 11
- SpringBoot 2.7.17
- Spring Data JPA
- QueryDsl

### build tool

- gradle

### 버전 관리

- git, github

## 요구사항

- 인증, 인가 처리 (구현 기술은 아직 구상중..)
- 회원이 판매하고자하는 아이템을 사진과 함께 올릴 수 있다.
- 댓글, 대댓글 기능
- 관리자 계정이 따로 존재하며 해당 계정을 통해 회원의 게시글을 삭제할 수 있다.
